Api-Huflit

Description

API Huflit page portal. NodeJs modules: request-promise, express, cheerio, body-parser,...

Guide Api

Get Cookie Jar (Login)

path: "/profile" "POST" data form as: {user: MSSV, pass: Mật Khẩu} response:

[

{"_jar": ...},

Tên_Sinh_Viên

]

or

"forgot user or pass"

Errors could happen due to an error or incorrect user or pass

Get Schedules

path: "/Schedules" "POST" data form as: {cookieJar: _jar} response:

{

Thu: Thứ,
Phong: Phòng,
MonHoc: Tên Môn Học,
MaLopHocPhan: Mã Lớp Học Phần,
LopHoc: Lớp,
TietHoc: Tiết Học,
GiaoVien: Tên Giáo Viên

  }

or []
Errors can occur when the "POST" incorrectly cookies or expires!

Change Password

path: "/ChangePass" "POST" data form as: {
  cookieJar: _jar,
  oldPass: oldPass,
  newPass: newPass
} response:

  "Cập nhật mật khẩu thành công..."

 or

  "Mật khẩu cũ không chính xác"

Errors can occur when the "POST" incorrectly cookies or expires!
